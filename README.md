# IntegrationHub.Api

Projeto criado para demonstrar conceitos de integração de sistemas corporativos utilizando APIs REST, mensageria e processamento assíncrono.

## Objetivos

- Integração entre sistemas
- Processamento assíncrono
- Resiliência
- Observabilidade
- Boas práticas de arquitetura

## Tecnologias previstas

- .NET
- SQL Server
- RabbitMQ
- Docker
- Swagger

## Fluxo

ERP → API → Banco → Fila → Processamento
